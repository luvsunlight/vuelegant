## 概述

> Divider 分割线

## props

|  参数  |           说明           |           类型           |  默认值   |
| :----: | :----------------------: | :----------------------: | :-------: |
|  type  | 分割线是水平的还是竖直的 | Enum(horizonal,vertical) | horizonal |
| align  |    分割线文字对齐方式    | Enum(left,center,right)  |  center   |
| dashed |     分割线是否是虚线     |         Boolean          |   false   |

!> 当 divide 的`type`为`vertical`时，不可以在其中嵌入文字
