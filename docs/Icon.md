# props

|  参数   |             说明              |  类型   | 默认值 |
| :-----: | :---------------------------: | :-----: | :----: |
|  name   |          图标的名称           | String  |  cog   |
|  scale  |          图标的大小           | String  |   1    |
|  color  |          图标的颜色           | String  |   1    |
| inverse |       图标是否颜色反转        | Boolean |  true  |
| pointer | 鼠标移至图标上是否更改 cursor | Boolean |  true  |
